# Auktiva Documentation

Welcome to the official documentation for **Auktiva** - a self-hosted, real-time auction platform built with Next.js.

## What is Auktiva?

Auktiva is an open-source auction platform that allows you to create and manage private or public auctions. It's designed to be self-hosted, giving you full control over your data and customization.

### Key Features

- **Real-time bidding** - Live updates using polling
- **Multiple auction types** - Private (invite-only), link-based, or open auctions
- **Flexible item management** - Custom end dates, starting bids, multiple images
- **Role-based access** - Owner, Admin, Creator, and Bidder roles
- **Anonymous bidding** - Optional bidder anonymity per auction or per bid
- **Notifications** - In-app notifications for outbids and auction events
- **Self-hosted** - Full control over your data with SQLite or Turso database
- **S3 compatible storage** - Local storage or any S3-compatible service for images
