# Invitations

Learn how to invite others to your auctions.

## Invitation Methods

### Email Invitations

1. Go to auction's **Invite** page
2. Enter email addresses
3. Click **"Send Invitations"**

### Invite Link

1. Go to auction's **Invite** page
2. Copy the invite link
3. Share via any channel

### Open Access

For open auctions, no invitation needed - anyone can join.

## Join Modes

| Mode            | Description                |
| --------------- | -------------------------- |
| **Invite Only** | Must use email invitations |
| **Link**        | Anyone with link can join  |
| **Open**        | Anyone can join            |

## Managing Invitations

### Revoking Invitations

1. Go to **Invite** page
2. Find the pending invitation
3. Click **"Revoke"**

<Callout type="warning">
  Revoking doesn't remove users who have already joined.
</Callout>

## Who Can Invite

- **Owner** and **Admin** can always invite
- Other roles can invite if "Member Can Invite" is enabled

## Accepting Invitations

### For New Users

1. Click the invitation link
2. Create an account
3. You're added automatically

### For Existing Users

1. Click the invitation link
2. Log in
3. Click **"Accept Invitation"**
