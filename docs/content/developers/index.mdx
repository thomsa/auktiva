# Developer Documentation

This section contains technical documentation for developers who want to set up, configure, deploy, or contribute to Auktiva.

## Tech Stack

Auktiva is built with modern technologies:

- **Framework**: [Next.js](https://nextjs.org/) 16 (Pages Router)
- **Database**: [Prisma ORM](https://www.prisma.io/) with SQLite or [Turso](https://turso.tech/)
- **Authentication**: [NextAuth.js](https://next-auth.js.org/) v4
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) + [DaisyUI](https://daisyui.com/)
- **Icons**: [Iconify](https://iconify.design/) (Tabler icons)
- **Storage**: Local filesystem or S3-compatible storage
- **Process Manager**: [PM2](https://pm2.keymetrics.io/) (production)

## Prerequisites

Before installing Auktiva, ensure you have:

- **Node.js** 18.x or higher
- **npm** 9.x or higher (comes with Node.js)
- **Git** (for cloning the repository)
